# Iris Flower Classification using Machine Learning

## Project Overview

This project uses Machine Learning to classify Iris flowers into three species:

* Iris-setosa
* Iris-versicolor
* Iris-virginica

The classification is performed using flower measurements:

* Sepal Length
* Sepal Width
* Petal Length
* Petal Width

## Objective

* Train a machine learning model using Scikit-learn
* Predict flower species from measurements
* Evaluate model accuracy using test data
* Understand basic classification concepts

## Technologies Used

* Python
* Pandas
* Scikit-learn
* VS Code

## Dataset

* Iris Dataset (Iris.csv)
